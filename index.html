Command Reference

All documented commands – organized by category.

Ping & Network
ping
ICMP echo to target.
ping 8.8.8.8
ping6
IPv6 ping.
ping6 2001:db8::1
ping_sweep
Ping sweep a /24 network.
ping_sweep 192.168.1.0/24
traceroute
Trace route to target.
traceroute example.com
whois
WHOIS lookup.
whois domain.com
dns / dig
DNS lookup (A, MX, etc).
dns google.com MX
location
IP geolocation.
location 8.8.8.8

nmap

nmap_os
OS detection.

nmap_vuln

Curl & Netcat
curl
HTTP GET request.

curl_post
HTTP POST with data.
curl_post url "key=value"
curl_head
HEAD request.
curl_head example.com
netcat
Connect to host:port.

nc_listen
Listen on port.
nc_listen 4444
nc_scan
Port scan with netcat.
nc_scan 10.0.0.1 20-80
SSH
ssh_add
Store SSH connection.
ssh_add name host user pass
ssh_list
List stored connections.
ssh_list
ssh_connect
Connect to stored host.
ssh_connect conn_id
ssh_exec
Execute command via SSH.
ssh_exec conn_id "ls -la"
